# מערכת ניהול ארנק כיתתי עם חנות 🏪

מערכת מלאה לניהול נקודות תלמידים וחנות כיתתית עם מערכת אישורים.

## תכונות המערכת

### למורים 👨‍🏫
- **ניהול תלמידים**: הוספה, עדכון יתרות, מעקב אחר כל התלמידים
- **ניהול חנות**: הוספת מוצרים, מחיקת מוצרים, קביעת מחירים
- **מערכת אישורים**: קבלת התראות על בקשות קנייה, אישור/דחיית קניות
- **היסטוריה מלאה**: צפייה בכל הקניות של כל התלמידים
- **הוספת מורים נוספים**: יכולת ליצור חשבונות מורה נוספים

### לתלמידים 👨‍🎓
- **ארנק דיגיטלי**: צפייה ביתרה האישית בממשק מעוצב
- **חנות אינטראקטיבית**: רכישת מוצרים מהחנות הכיתתית
- **מעקב קניות**: צפייה בהיסטוריית הקניות ובסטטוס האישורים
- **התראות**: מעודכנים על סטטוס הקנייה (ממתין/אושר/נדחה)

## התקנה והפעלה

### דרישות מקדימות
- Node.js מותקן במחשב
- חשבון MongoDB (MongoDB Atlas - חינם)

### שלבי התקנה

1. **התקנת חבילות**
```bash
cd "C:\Users\przyy\Desktop\ארנק כתתי"
npm install
```

2. **הגדרת קובץ סביבה**
צור קובץ בשם `.env` בתיקיית הפרויקט עם התוכן הבא:
```
MONGO_URI=mongodb+srv://[USERNAME]:[PASSWORD]@[CLUSTER].mongodb.net/classroom?retryWrites=true&w=majority
ADMIN_PASSWORD=1234
PORT=3000
```
החלף את [USERNAME], [PASSWORD] ו-[CLUSTER] עם הפרטים מ-MongoDB Atlas שלך.

3. **הפעלת השרת**
```bash
node server.js
```

4. **גישה למערכת**
פתח דפדפן וגש ל: `http://localhost:3000`

## הוראות שימוש

### התחברות ראשונית
- **מורה**: לחץ על "כניסת מורה" והקלד את הסיסמה (ברירת מחדל: 1234)
- **תלמיד**: לחץ על "כניסת תלמיד" והקלד את קוד התלמיד שלך

### יצירת משתמשים חדשים (מורים בלבד)
1. היכנס כמורה
2. בחר האם ליצור תלמיד או מורה נוסף
3. למלא את הפרטים הנדרשים
4. לחץ על "צור תלמיד" או "צור מורה"

### ניהול החנות (מורים)
1. לחץ על כפתור "ניהול חנות 🏪"
2. **טאב מוצרים**: הוסף מוצרים חדשים עם שם, מחיר ותיאור
3. **טאב בקשות ממתינות**: אשר או דחה קניות של תלמידים
4. **טאב היסטוריה**: צפה בכל הקניות שבוצעו

### קנייה בחנות (תלמידים)
1. לחץ על "🛒 עבור לחנות"
2. עיין במוצרים הזמינים
3. לחץ על "קנה עכשיו" על המוצר הרצוי
4. המתן לאישור המורה
5. בטאב "הקניות שלי" תוכל לראות את סטטוס הקנייה

### תהליך אישור קנייה
1. תלמיד מבצע בקשת קנייה
2. המורה מקבל התראה (תג אדום על טאב "בקשות ממתינות")
3. המורה בוחן את הבקשה ומאשר/דוחה
4. אם אושר - הנקודות מורדות אוטומטית מיתרת התלמיד
5. התלמיד מקבל עדכון על הסטטוס

